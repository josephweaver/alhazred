# Alhazred

Alhazred is a persistent AI-mediated tabletop roleplaying campaign and experimental narrative simulation framework built around the 24-System.

## Core Concepts

- Narrative-first interaction
- Persistent world state
- AI-assisted refereeing
- Git-backed campaign continuity
- Structured YAML runtime state
- Horror, mystery, and survival gameplay

## Repository Structure

```text
rules/       Core mechanics and resolution systems
lore/        Canonical world information
characters/  Persistent player and NPC files
state/       Runtime campaign state
sessions/    Session summaries and history
scenarios/   Adventures and encounter modules
ai/          AI referee prompts and protocols
```

## Philosophy

The AI is not the author.

The AI acts as:
- referee,
- simulator,
- narrator,
- and state manager.

The repository acts as the canonical memory and persistent continuity layer for the campaign world.
