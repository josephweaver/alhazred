# Alhazred Startup Guide

## Purpose

This repository contains the canonical setting, philosophy, rules, and narrative architecture for the Alhazred runtime.

It represents:
- stable setting truth,
- long-term continuity,
- system philosophy,
- AI Guide behavior,
- and canonical world structure.

This repository should be treated as the primary authority for:
- lore,
- tone,
- philosophy,
- and narrative identity.
